On-Page SEO Checklist

Title tags, meta descriptions, headings, and internal links.

Before You Publish

Use this checklist to ensure every page is optimized for search. Based on how top brands like Nykaa and Amazon structure their on-page SEO.

Title Tag

  • Include primary keyword near the start (ideally in the first 60 characters).
  • Keep under ~60 characters so it doesn’t truncate in SERPs.
  • Make it unique per page and compelling for clicks (benefit or brand).
  • Format: [Primary Keyword] – [Secondary/Brand] or [Brand] | [Topic].

Meta Description

  • Summarize the page and include primary or secondary keyword naturally.
  • Keep under ~155–160 characters.
  • Add a clear CTA or benefit (e.g. "Learn how", "Buy now", "Get the guide").
  • Unique per page; don’t duplicate across similar pages.

URL Structure

  • Short, readable, and descriptive. Include keyword if it fits naturally.
  • Use hyphens; avoid special characters and long strings of IDs.
  • Prefer lowercase. Avoid unnecessary parameters in the canonical URL.

Heading Structure

  • One H1 per page (main topic or primary keyword).
  • Logical hierarchy: H1 → H2 → H3. Don’t skip levels.
  • Use H2/H3 for key subtopics; include secondary keywords where natural.
  • Headings should outline the content; avoid keyword stuffing.

Internal Linking

  • Link to related category pages, pillar pages, and key product/content pages.
  • Use descriptive anchor text (not "click here").
  • Ensure important pages get internal links from multiple places (home, category, blog).
  • Fix broken internal links and redirect old URLs when you change structure.

Quick Checklist Summary

  • ✓ Title tag: keyword, <60 chars, unique, compelling
  • ✓ Meta description: keyword, <160 chars, CTA, unique
  • ✓ URL: short, readable, hyphenated
  • ✓ H1: one per page, primary topic
  • ✓ H2/H3: logical hierarchy, subtopics
  • ✓ Internal links: relevant anchors, key pages linked
